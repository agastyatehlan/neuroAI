What Dimag does

Dimag 1.0 is a compact inference and control module that provides perception, decision-making, and behaviour policy execution so devices can act with human-like adaptiveness while remaining safe and auditable.

Plug-and-play
Supports SPI / I2C / UART / CAN — easy integration into robotics and automotive stacks.
Multiple Modes
Autonomous, Assist, Safe-Emergency, Simulation. Switch modes programmatically or via a secure channel.
Use Cases
Drones (navigation & obstacle avoidance), Robotics (manipulation & perception), Customer service (speech + intent), Specialized military-grade deployments with hardened features.
Safety & Security
Encrypted models, signed firmware updates, runtime constraints, and clear audit logs to support safe operation.
Tech specs (summary)
  • Low-power NPU + microcontroller
  • On-device model runtime (quantized)
  • Interfaces: SPI, I2C, UART, CAN
  • Secure boot & signed updates
Integration

We provide SDKs and examples to connect Dimag to sensors, cameras, and control loops. Example stacks for drone autopilots and ROS are available.
